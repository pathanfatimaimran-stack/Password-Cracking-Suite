# Password Cracking & Credential Attack Suite

## 📌 Description
This project simulates a dictionary-based password attack and tests password strength using hashing.

## 🎯 Objectives
- Detect weak passwords
- Perform dictionary attack
- Compare hashed values

## 🛠️ Tools Used
- Python
- hashlib

## ▶️ How to Run
python3 password_cracker.py

## 📊 Output
- Shows cracked password if weak
- Displays strong password message
